# Ians_NotePlan_Plugins
NotePlan plugins created with AI

## Plugins

| Plugin | Description |
|--------|-------------|
| [iansoper.AmieSync](./iansoper.AmieSync) | Syncs meeting notes, summaries, action items, and audio from Amie into NotePlan daily and project notes |
| [iansoper.NotionMeetingSync](./iansoper.NotionMeetingSync) | Syncs Fireflies and native Notion meeting recordings into NotePlan as markdown notes with tasks |
